# T-te-037: Size-threshold rotation-safe JSONL writer
**Status:** done

`createInMemoryTraceSink(name, rotateBytes)`:
- Tracks `activeBytes` per writing file
- When `activeBytes + lineBytes > rotateBytes`, seals the current file under `<name>.<rotations>` and opens a fresh file
- `files()` exposes every file
- `snapshot()` returns all lines across rotations

Test: "rotates when size threshold crossed, preserving prior bytes" writes 20 events with a 100-byte threshold and asserts `rotations > 0`, `fileKeys.length > 1`, and `totalLines === 20`.

Acceptance R5 AC-4 met.
